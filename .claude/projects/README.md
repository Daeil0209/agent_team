# Project Output Root Policy Pointer

This file is governance documentation only. It does not make `.claude/projects/`
an output root.

All task-created output files belong under the active repository root:
`projects/<project-folder>/`.

This includes deliverables, reports, evidence, logs, screenshots, exports,
datasets, tool caches, intermediate files, and every nested output they create.

Do not write generated work products into governance, hook, role, skill,
reference, runtime-state, home, temp, or incidental tool-current directories.
